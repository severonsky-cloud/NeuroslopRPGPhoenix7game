# Как запустить текущий билд Phoenix7

## Коротко

На Windows в корне репозитория нажми:

```text
00_START_HERE_CURRENT_BUILD_WINDOWS.bat
```

Это главный файл для обычного теста игры.

Он запускает локальный сервер и сам открывает:

```text
http://localhost:8000/v3l.html
```

## Очень важно

Не открывай `v3l.html` двойным кликом. Игра использует ES modules, поэтому ей нужен локальный сервер.

Не используй для обычного теста старые файлы:

```text
game.html
real3d.html
act1_*.html
game_v3.html
game_v3i.html
v3k.html
```

Они оставлены как старые прототипы/совместимые входы, но текущая линия разработки — `v3l.html`.

## Что делать после моих обновлений

1. Закрой старую вкладку игры.
2. Запусти `00_START_HERE_CURRENT_BUILD_WINDOWS.bat`.
3. Когда откроется браузер, нажми `Ctrl+F5`, чтобы сбросить кэш.
4. Нажми старт/продолжить в игре.

## Где смотреть новые изменения живого мира

После запуска:

```text
F1 — тестовый телепорт
J — журнал событий
M — карта
E — говорить с NPC/караваном рядом
1–7 — оружие
V — прицел
B — альтернативная атака
```

Для проверки живого мира:

1. Нажимай `F1`, пока не попадёшь на Красную дорогу или к Форту Заря.
2. Ищи караваны, патрули, бандитов, Жужжер и рыцарей.
3. Открывай `J` — там появляются события торговли, засад и фракционной жизни.
4. Если караван атаковали, стреляй по налётчикам: они созданы как hostile monsters, поэтому текущее оружие должно их поражать.

## Ручной тест засады через консоль браузера

Открой DevTools / Console и вставь:

```js
const lw = PHX_V3_ENGINE.livingWorld;
const caravan = lw.agents.find(a => a.userData.id === 'red_clay_caravan').userData;
lw.spawnCaravanAmbush(caravan, 'bandits', 'ручной тест засады');
```

Диагностика живого мира:

```js
PHX_V3_ENGINE.livingWorld.diagnostics()
```

```text

Реальный launcher: START_GAME_V3L_WINDOWS.bat
Реальный HTML entrypoint: v3l.html
Реальный runtime: src/game/v3/
Главная сборка движка: src/game/v3/main.js
```
