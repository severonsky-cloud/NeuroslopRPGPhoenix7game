# Как запускать Phoenix7 локально

GitHub Pages может не работать на приватном репозитории без подходящего тарифа. Поэтому самый надёжный запуск — локально.

## Вариант 1: Python-сервер

1. Скачай репозиторий: Code → Download ZIP.
2. Распакуй папку.
3. Открой терминал/PowerShell внутри этой папки.
4. Запусти:

```bash
py -m http.server 8000
```

Если `py` не работает:

```bash
python -m http.server 8000
```

5. Открой в браузере:

```text
http://localhost:8000/real3d.html
```

## Что открывать

- `real3d.html` — настоящий Three.js 3D-слой 2.0F.
- `index.html` — fallback-ядро, если CDN/Three.js не загрузился.

## Почему нельзя просто двойным кликом

`real3d.html` использует Three.js как ES module через CDN. Браузеру часто нужен локальный сервер, иначе модули и ассеты могут не загрузиться.

## Управление

- WASD — ходьба
- Shift — быстрее
- мышь — обзор после клика по экрану
- E — говорить/действие
- Space — удар
- J — Книга дорог
- M — карта
- I — инвентарь
- F1 — debug travel между важными точками
