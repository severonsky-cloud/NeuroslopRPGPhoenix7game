Phoenix7 2.0O / audio fallback lab

Звук Феникса 7

Тест звукового слоя: сейчас работает procedural WebAudio fallback, потом эти же кнопки смогут брать WAV из public/assets/audio/. Игра не должна ломаться, если файла нет.

Audio: ждёт первого клика

import { PhoenixAudio } from './src/phoenix_audio.js';
const audio = new PhoenixAudio();
await audio.init();
audio.play('questUpdate');

Когда загрузим WAV-файлы в public/assets/audio/, этот менеджер сначала попробует файл из manifest, а если файла нет — сыграет procedural fallback.
