Как запускать локально

py -m http.server 8000

Главная игра:

http://localhost:8000/game.html

Меню билдов:

http://localhost:8000/play.html

GitHub Pages для приватного репозитория может быть недоступен без подписки, поэтому локальный сервер — основной способ разработки.
