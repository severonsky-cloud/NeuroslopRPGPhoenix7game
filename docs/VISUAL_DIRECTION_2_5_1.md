# Phoenix7 2.5.1 — Visual Direction

## Главная цель

Вернуть визуальное направление к клодовскому PC-билду, но сохранить наши игровые идеи:

```text
красивый атмосферный frontier RPG
+ Порт Рейчел
+ Форт Заря
+ дороги
+ двери и интерьеры
+ NPC внутри зданий
+ руки игрока
+ фазовый удар рукой
+ арсенал
+ лут
+ первый акт Герды
```

## Что изменено в game.html

`game.html` получил visual pass:

- тёплый медный skybox;
- два солнца;
- тёмный ретро-RPG ground;
- псевдо-3D raycast view;
- грязный экран / scanlines;
- vignette;
- grain/noise;
- более атмосферные руки;
- glow для фазового удара;
- более заметные силуэты оружия;
- minimap оставлена как dev-help.

## Почему не возвращаем старый монолит напрямую

Старый клодовский билд был красивее местами, но он не должен снова превращать проект в хаос из HTML.

Правильный путь:

```text
сохраняем один главный game.html
берём визуальные идеи из клодовского билда
переписываем их безопасно внутрь стабильного runtime
```

## Визуальный ориентир

- low-poly / early 2000s PC RPG;
- Morrowind-like sense of place;
- тёплый апокалиптический свет;
- грязные здания;
- медные/красные оттенки Феникса 7;
- странная frontier sci-fi/fantasy;
- не глянцево;
- не mobile casual;
- не чистый tech demo.
