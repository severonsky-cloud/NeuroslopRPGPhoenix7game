# Phoenix7 — Roadmap to 2.5

## Цель версии 2.5

2.5 должна быть первым цельным playable vertical slice, а не набором отдельных HTML-кандидатов.

Формула 2.5:

```text
один запуск: game.html
один первый акт
живой Порт Рейчел
доступные здания с дверями
NPC внутри и снаружи
видимые руки игрока
арсенал
фазовый удар рукой
враги и лут
сохранение
звук
модельный fallback
```

## Главное правило разработки

Новые игровые правки идут сначала в:

```text
game.html
```

Лаборатории можно использовать как доноров, но игрок не должен бегать по разным HTML.

## 2.4.1 — Stabilization Hotfix

Цель: игра должна стабильно открываться.

Tasks:

- Проверить `game.html` на чистом локальном сервере.
- Убрать любые зависимости, из-за которых главный запуск даёт чёрный экран.
- Оставить видимую ошибку, если JS падает.
- Проверить сохранение/загрузку.
- Проверить, что `play.html` и README указывают на `game.html`.

Definition of Done:

```text
http://localhost:8000/game.html открывается и даёт начать игру
```

## 2.4.2 — Doors and Interiors Pass

Цель: здания не должны быть непроходимыми коробками.

Tasks:

- Таможня: вход, интерьер, Рина внутри.
- Канцелярия: вход, интерьер, Оран внутри.
- Дом Герды: вход, интерьер, Герда внутри.
- Рынок: лавка/навес без запирания NPC.
- Красный Узел: вход/внутренний двор.
- Лагерь проводников: открытый доступ.

Definition of Done:

```text
К каждому важному NPC можно подойти без F1/телепорта
```

## 2.4.3 — Hands and Arsenal Polish

Цель: руки и оружие должны ощущаться как часть персонажа.

Tasks:

- Улучшить вид рук.
- Отдельная поза для кулаков.
- Отдельная поза для фазового удара.
- Отдельные силуэты для бастарда и шпаги.
- Реликтовое огнестрельное должно иметь разные силуэты.
- Ammo UI сделать читаемее.
- Добавить короткий hit feedback.

Definition of Done:

```text
1–7 визуально отличаются и понятно, чем игрок атакует
```

## 2.4.4 — Quest Loop Cleanup

Цель: первый акт должен быть понятен без подсказок из чата.

Tasks:

- Рина даёт ясную первую цель.
- Оран объясняет регистрацию.
- Герда открывает список подготовок.
- Доска Герды показывает 4 подготовки.
- После 4/4 открывается проводник.
- Журнал пишет, что произошло.

Definition of Done:

```text
Игрок может пройти первый акт до проводника без внешней инструкции
```

## 2.4.5 — Audio Integration

Цель: звук подключён к главному билду, но не ломает игру.

Tasks:

- UI click.
- Start game sound.
- Book/map/inventory sounds.
- Quest update.
- Quest complete.
- Attack sounds.
- Phase strike sound.
- Port ambience.
- Road/Fort ambience placeholder.

Definition of Done:

```text
Если WAV отсутствуют, procedural fallback играет и билд не падает
```

## 2.4.6 — Model Fallback Integration

Цель: главный билд умеет подменять procedural NPC на GLB, когда GLB появятся.

Tasks:

- Gerda model slot.
- Oran model slot.
- Rina model slot.
- Wandering knight model slot.
- Green/Blue faction props.
- Enemy placeholder slots.

Definition of Done:

```text
GLB можно добавить в manifest, и game.html не надо переписывать
```

## 2.5 — First Playable Vertical Slice

2.5 считается готовой, когда:

- Запуск один: `game.html`.
- Есть начало в Порту Рейчел.
- Есть 3 доступных здания с NPC внутри.
- Есть первый маршрут до Форта Заря.
- Есть Герда и 4 задачи.
- Есть оружие 1–7.
- Есть phase strike рукой.
- Есть враги и лут.
- Есть сохранение.
- Есть базовый звук.
- Есть хотя бы частичный model fallback.
- Игрок может пройти от старта до проводника.
