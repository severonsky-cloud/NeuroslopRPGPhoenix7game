# Phoenix7 2.0N — Integrated Character Creator → Act 1 Loop

Файл: `act1_integrated.html`

## Что это

Нормальный стартовый поток:

```text
живой редактор персонажа
→ сохранение героя
→ совместимый сейв для Act 1 Loop
→ автоматический запуск `act1_loop.html`
→ автоклик Continue
→ игрок в Порту Рейчел
```

## Почему сделано отдельным файлом

`act1_loop.html` уже рабочий и большой. Чтобы не сломать его, 2.0N сделан как безопасная обёртка:

- создаёт персонажа;
- сохраняет `phx2l_character`;
- создаёт совместимый сейв `phx2m_act1_loop`;
- открывает `act1_loop.html` в iframe;
- нажимает кнопку `Продолжить` внутри iframe.

## Что переносится из Character Creator

- имя;
- раса;
- предыстория;
- стартовые HP/ST/resonance;
- стартовые кредиты;
- цвета и рост сохраняются во флаги сейва.

## Запуск

```bash
py -m http.server 8000
```

Открыть:

```text
http://localhost:8000/act1_integrated.html
```

## Следующий шаг

Если этот поток стабилен, можно сделать `act1_integrated.html` главным запуском в `play.html`, а позже перенести код редактора внутрь `act1_loop.html` уже без iframe.
