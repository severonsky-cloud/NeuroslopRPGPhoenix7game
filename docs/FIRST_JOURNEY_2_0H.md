# Phoenix7 2.0H — First Journey Build

Файл: `journey.html`

Цель: сделать первый полноценный игровой маршрут, а не просто тестовый мир.

## Игровой путь

```text
Создание персонажа
→ пробуждение в Порту Рейчел
→ разговор с Риной в таможне
→ проверка Дорожного навеса
→ Старый маяк
→ бродячий рыцарь / караван
→ ворота Форта Заря
→ регистрация у Орана Тива
→ первая встреча с Гердой
```

## Что есть

- Настоящий Three.js 3D слой.
- Создание персонажа: имя, раса, предыстория.
- Несколько стадий первого квеста.
- Дорожные POI и авто-обновление Книги дорог.
- NPC и диалоги.
- Первый враг/опасность на дороге.
- Караван как движущийся объект.
- Инвентарь, журнал, карта.
- F1 debug travel по важным точкам.

## Запуск

```bash
py -m http.server 8000
```

Затем:

```text
http://localhost:8000/journey.html
```
