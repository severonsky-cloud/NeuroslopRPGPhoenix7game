# Phoenix7 2.0K — Director Journey Pass

Файл: `journey_director.html`

## Задача

Сделать первый маршрут менее похожим на технический тест и больше похожим на режиссёрский вертикальный срез: читаемый путь, больше ориентиров, больше декораций, больше постановки.

## Что добавлено

- Более плотный Порт Рейчел.
- Больше ящиков, досок и причальных объектов.
- Маршрутные знаки:
  - к Форту Заря;
  - к Навесу;
  - к Орденам.
- Больше фонарей по дороге.
- Больше сухих деревьев и камней как силуэты мира.
- Улучшена структура стартового маршрута:
  - Порт;
  - Рина;
  - Навес;
  - Маяк;
  - Рыцарь;
  - Ворота;
  - Оран;
  - Герда.
- Сохранена вся квестовая цепочка из `journey_visual.html`.

## Запуск

```bash
py -m http.server 8000
```

Открыть:

```text
http://localhost:8000/journey_director.html
```

## Роль в производстве

- `journey.html` — стабильный маршрут.
- `journey_visual.html` — визуальный маршрут.
- `journey_director.html` — режиссёрский кандидат на главный первый vertical slice.
