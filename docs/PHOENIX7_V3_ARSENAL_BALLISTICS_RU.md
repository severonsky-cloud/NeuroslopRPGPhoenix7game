## Альтернативные атаки

Клавиша:

```text
B
```

Если оружие поддерживает:

```text
пистолет/карабин/Брен → удар рукоятью или прикладом
M1 Гаранд → штык
окопный дробовик → штык / приклад
мечи → тяжёлая атака, если есть профиль
```

## Мушка / прицельный режим

Клавиша:

```text
V
```

Один раз нажать:

```text
включает прицельный режим
сужает FOV
стабилизирует оружие
ставит более строгую мушку
```

Повторное нажатие возвращает обычный обзор.

## Баллистика

Теперь выстрел не является просто медленным шариком.

Система `BallisticSystem` считает:

```text
направление камеры
spread
навык Огнестрел
aim mode
скорость пули
падение траектории
дистанцию
hit radius цели
```

Визуально создаётся:

```text
быстрый tracer line
impact sparks
hit marker
```

## Инвентарь

Инвентарь показывает:

```text
иконки оружия
иконки патронов
патроны отдельными ресурсами
weapon ammo type
куклу персонажа
левый/правый набор оружия
```

## Управление

```text
ЛКМ / Space — основная атака
B — приклад / штык / альтернативная атака
V — мушка / прицел
O — покупка патронов
I — инвентарь
Tab — сменить левый/правый набор оружия
K — персонаж
P — фазовая магия
```
