## Зачарование оружия

Клавиша:

```text
Y
```

Открывает окно зачарования активного оружия.

Зачарования:

```text
Фазовая кромка
Стеклянный укус
Синий заряд
Чёрная клятва
Ледяная руна
```

Работают и на ближнем оружии, и на огнестреле, если тип подходит.

Примеры:

```text
Фазовая кромка — ближнее оружие, дополнительный фазовый урон
Синий заряд — огнестрел, меньше spread и меньше шанс осечки
Чёрная клятва — сильнее, но быстрее портит оружие
Ледяная руна — больше stagger
Стеклянный укус — шанс дополнительного осколочного урона
```

## Ненадёжный старый огнестрел

Каждое огнестрельное оружие имеет:

```text
loaded
clipSize
condition
jammed
reloadT
```

Оружие может дать осечку.

Шанс осечки зависит от:

```text
типа оружия
состояния оружия
зачарований
```

Клавиша:

```text
R
```

Если оружие пустое — перезарядка.
Если осечка — устранение осечки.

## Патроны

Патроны хранятся отдельно:

```text
.45 revolver
rifle clips
Bren magazines
shot shells
phase cells
```

Магазин патронов:

```text
O
```

Лут с врагов:

```text
road loot
raider loot
elemental loot
bandit loot
```

При убийстве врага система может добавить патроны в инвентарь и показать это в HUD.

## Интерактивная экипировка

В инвентаре появились кнопки:

```text
В левый набор
В правый набор
Надеть: слот
```

Левая/правая рука остаются двумя наборами оружия:

```text
Tab — переключить набор
```

## Альтернативные атаки

Клавиша:

```text
B
```

Оружие может использовать:

```text

штык
тяжёлую атаку
```

## Баллистика и зачарования

Баллистика теперь принимает модификаторы:

```text
damageScale
spreadMul
jamMul
conditionWearMul
```

Это позволяет зачарованиям влиять на огнестрел:

```text
сильнее урон
меньше spread
выше/ниже шанс осечки
быстрее/медленнее износ
```

## Управление

```text
ЛКМ / Space — основная атака
R — перезарядить / устранить осечку
B — приклад / штык / альтернативная атака
V — мушка / прицел
Y — зачарование оружия
O — купить патроны
I — инвентарь
Tab — сменить левый/правый набор оружия
```
