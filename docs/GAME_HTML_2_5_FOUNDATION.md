# Phoenix7 — game.html 2.5 Foundation

## Главный запуск

```text
http://localhost:8000/game.html
```

## Что изменено

`game.html` больше не является wrapper/iframe и не зависит от лабораторных HTML.

Он стал единым самодостаточным foundation-билдом:

```text
no iframe
no external JS modules
no CDN imports
single playable file
```

## Почему так

Ранее главный запуск мог ломаться из-за:

- внешнего Three.js CDN;
- ES module import;
- зависимости от `act1_arsenal_world.html`;
- путаницы между несколькими HTML-кандидатами.

Теперь `game.html` можно тестировать как один файл на локальном сервере.

- Таможня: Рина внутри.
- Канцелярия: Оран внутри.
- Дом Герды: Герда внутри.
- Грязный рынок.
- Красный Узел.
- Лагерь проводников.

## Управление

```text
WASD — движение
Shift — бег
мышь / стрелки ← → — обзор
ЛКМ или Space — атака
E — говорить / обыскать
R — блок
1 — кулаки
2 — фазовый удар рукой
3 — бастард
4 — шпага
5 — Кольт 1917
6 — M1 Гаранд
7 — Брен
J — журнал
I — инвентарь
Q — задачи Герды
F1 — телепорт по контрольным точкам
Esc — закрыть панель / выйти из pointer lock
```
