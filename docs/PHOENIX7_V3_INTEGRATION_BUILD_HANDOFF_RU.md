# Phoenix7 v3 — интеграционный билд (handoff для продолжения)

**Ветка:** `claude/v3-integration-build` (на origin). **Дата:** 2026-06-20.

## Что внутри (всё проверено вместе, build = 76 модулей, без ошибок)

Сводит в один играбельный билд:

- **main `b5d745b`** — v3M1 (8 поселений, тракт Порт→Форт) + v3M2A (создание персонажа: профиль, 8 рас, расовые руки/тело, Q-способности).
- **Фикс Брена `0e66d2f`** — камеру больше не закручивает (roll всегда 0, pitch/отдача ограничены); компактная модель Брена, не закрывает центр экрана.
- **v3N1 день-ночь** — цикл суток (клавиша `T` — ускорение), солнце/туман/луна/звёзды по времени; распорядок жителей (день — работа, ночь — по домам); события суток в журнале.
- **v3N2 точки интереса** — 7 руин/святилищ/останков между поселениями; находишь вблизи → запись в журнал «Находки» + маркер на карте (fog-of-war: до находки скрыто).

## Как запускать / показывать

Локальный репозиторий сейчас на этой ветке, поэтому обычный лаунчер уже показывает полный билд:

```
START_GAME_V3L_WINDOWS.bat → http://localhost:8000/v3l.html   (Ctrl+F5)
```

Совместимость: `v3k.html`. Проверки: `npm.cmd run build`, `npm.cmd run test:v3m2a`.

Управление: мышь — обзор, движение WASD, ЛКМ — атака, `R` — перезарядка, `V` — прицел, `B` — приклад/штык, `T` — время суток, `M` — карта, `J` — журнал, `E` — говорить, `F2` — телепорт по поселениям, `F8` — третье лицо, `F9/F10/F11` — качество.
