# Phoenix7 2.0I — Visual Journey Pass

Файл: `journey_visual.html`

Цель: поднять визуал маршрута Порт Рейчел → Форт Заря без риска сломать рабочий `journey.html`.

## Что добавлено

- Более плотная сцена Порта Рейчел.
- Вода с визуальными полосами.
- Больше ящиков и досок у причала.
- Больше фонарей вдоль дороги.
- Деревья/сухие конусы/камни как силуэтный мусор мира.
- Лёгкие атмосферные облака/туман.
- Улучшенные подписи и ориентиры.
- Та же квестовая цепочка из 2.0H.

## Что проверять

Открыть:

```text
http://localhost:8000/journey_visual.html
```

Проверить:

1. Новая игра → создание персонажа.
2. Быстрый старт.
3. Видно ли Порт Рейчел лучше, чем в 2.0H.
4. Работает ли маршрут Рина → Навес → Маяк → Рыцарь → Ворота → Оран → Герда.
5. Не падает ли FPS.

## Дальше

2.0J должен быть не ещё одним визуальным костылём, а Asset Layer:

- подключение WAV-пака;
- подключение GLBLoader;
- папки `public/assets/models/` и `public/assets/audio/`;
- fallback на procedural models;
- первые настоящие модели: Рина, Оран, Герда, бродячий рыцарь, зелёный/синий послушник.
