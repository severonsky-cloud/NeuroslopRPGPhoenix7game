# Phoenix7 2.5 — Clean Rebuild Plan

## Главный запуск

Игра всегда запускается только отсюда:

```text
http://localhost:8000/game.html
```

`play.html`, `act1_*.html`, `model_lab.html`, `audio_lab.html` — больше не считаются игровыми входами. Это архив и доноры.

## Почему нужен rebuild

Текущий `game.html` ушёл не туда:

- raw WebGL открылся, но сцена не читается как игра;
- горизонт/камера уехали;
- объектов и маршрута почти не видно;
- билд стал техническим тестом, а не playable RPG;
- попытки быстро менять 2D / pseudo-3D / raw WebGL ломают цельность.

Вывод: нужен чистый rebuild 2.5, а не очередной hotfix поверх сломанного файла.

## Цель 2.5

Сделать первый цельный playable vertical slice:

```text
единый game.html
+ настоящий 3D-вид
+ старт в Порту Рейчел
+ доступные здания с дверями
+ Рина / Оран / Герда внутри помещений
+ путь до Форта Заря
+ первый акт Герды
+ руки игрока
+ фазовый удар рукой
+ оружие и боезапас
+ враги и лут
+ save/load
+ визуальное направление как у клодовского PC-билда
```

## Визуальная цель

Не чистый технодемо-лаб.

Визуальный ориентир:

- клодовский PC-билд;
- low-poly / early 2000s RPG;
- Morrowind/Gamebryo mood;
- тёплое медное небо;
- грязный frontier sci-fi/fantasy;
- Порт Рейчел с силуэтами, мачтами, рынком;
- Форт Заря как читаемый ориентир;
- тёмные интерьеры с цветовой разницей;
- не mobile casual;
- не пустой WebGL-шаблон.

## Техническое правило

Не писать огромный хаотичный `game.html` вручную.

Новая структура:

```text
game.html                  # один запуск
src/game/main.js            # старт игры
src/game/engine.js          # сцена, камера, рендер, цикл
src/game/world.js           # мир, здания, двери, коллизия
src/game/act1.js            # первый акт, квесты, NPC
src/game/combat.js          # руки, оружие, урон, лут
src/game/ui.js              # HUD, журнал, инвентарь, задачи
src/game/data.js            # данные NPC/items/quests
```

Игрок всё равно открывает только `game.html`.

## Engine decision

Для 2.5 не строить raw WebGL руками с нуля.

Причина: слишком легко получить сломанную камеру/матрицы/пустой горизонт.

Использовать один из вариантов:

### Preferred

Three.js как локальный или стабильный внешний модуль.

### Fallback

Если CDN не доступен, показывать честный экран ошибки с инструкцией, а не менять жанр игры на 2D.

## Этап 0 — Stop the bleeding

Цель: остановить хаос.

Tasks:

- Не делать новые `act1_*.html` как игровые билды.
- Не переписывать `game.html` в 2D fallback.
- Не делать ещё один raw WebGL эксперимент поверх текущего.
- Зафиксировать текущий файл как неудачный `full3d attempt`.
- Новый rebuild вести отдельными понятными шагами.

Definition of Done:

```text
Есть этот план, дальше правки идут по этапам.
```

## Этап 1 — Clean boot

Цель: `game.html` стабильно открывает сцену.

Минимум:

- стартовый экран;
- кнопка Quick Start;
- 3D canvas;
- камера от первого лица;
- простой ground;
- видимый sky/fog;
- HUD;
- красный overlay ошибки, если JS падает.

Definition of Done:

```text
http://localhost:8000/game.html открывается, игрок видит землю/небо и может двигаться.
```

## Этап 2 — Movement and collision

Цель: базовое передвижение без багов.

Tasks:

- WASD;
- мышь/pointer lock;
- Shift бег;
- простая capsule/circle collision;
- стены останавливают;
- двери не останавливают;
- F1 debug teleport только как dev tool.

Definition of Done:

```text
Игрок не проваливается, не летит, не застревает в пустоте.
```

## Этап 3 — Port Rachel blockout

Цель: построить читаемый Порт Рейчел.

Объекты:

- причал / портовый силуэт;
- таможня;
- рынок;
- дорога на Форт Заря;
- старый маяк;
- Красный Узел на стороне.

Важное:

- здания не цельные кубы;
- двери видны;
- входы реально проходные;
- внутри есть пространство.

Definition of Done:

```text
Игрок понимает, где Порт, где дорога, где таможня.
```

## Этап 4 — Interiors and NPC reachability

Цель: исправить главную проблему NPC.

NPC:

- Рина внутри таможни;
- Оран внутри канцелярии;
- Герда внутри дома;
- торговец на рынке;
- контрабандист у Красного Узла;
- Сава в лагере проводников.

Rules:

- каждого NPC можно достичь пешком;
- не нужен F1;
- NPC не стоит в стене;
- у двери есть визуальный маркер;
- E открывает диалог.

Definition of Done:

```text
Рина, Оран, Герда доступны без телепорта.
```

## Этап 5 — First Act quest chain

Цель: первый акт становится понятным.

Flow:

```text
1. Старт в Порту Рейчел.
2. Поговорить с Риной.
3. Проверить Дорожный навес.
4. Идти к Форту Заря.
5. Поговорить с Ораном.
6. Найти Герду.
7. Получить 4 подготовки:
   - дорога;
   - рынок;
   - фракции;
   - контрабанда.
8. Сдать Герде.
9. Поговорить с Савой.
10. Подготовка к Мёртвой Саванне.
```

Definition of Done:

```text
Игрок может пройти Act 1 до Савы без подсказок из чата.
```
