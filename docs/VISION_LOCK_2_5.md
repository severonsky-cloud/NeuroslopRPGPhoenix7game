# Phoenix7 2.5 — Vision Lock

## Главная цель

Phoenix7 2.5 — это не технический демо-лаб, не 2D fallback и не пустой WebGL-тест.

Цель:

```text
полноценная 3D RPG-игра
+ атмосферный PC-билд в духе старого клодовского прототипа
+ медленное исследование открытого мира
+ первый акт на берегу океана
+ смесь Morrowind и Medal of Honor
+ космоопера Phoenix7
```

## Главный запуск

Игрок всегда открывает только:

```text
http://localhost:8000/game.html
```

Остальные HTML — архивы, доноры, лаборатории, но не игровой вход.

## Важный запрет

Нельзя снова уводить главный билд в:

- 2D top-down;
- pseudo-3D без настоящего мира;
- пустой raw WebGL с горизонтом и без объектов;
- отдельные act1-файлы как главный запуск;
- чистый технодемо-лаб без атмосферы.

## Визуальный ориентир

Брать за основу старый клодовский PC-билд:

- низкополигональная 3D-сцена;
- грязный атмосферный экран;
- ощущение раннего 2000-х PC RPG;
- тёплый мрак;
- медное/красное небо;
- пыль, красная глина, влажный океанский берег;
- странная космоопера, но с военными реликтами;
- не глянцево;
- не mobile casual;
- не стерильный tech demo.

## География / биом

Феникс 7 должен ощущаться как океанский берег, от которого дорога постепенно уходит в Мёртвую Саванну.

Визуальные источники:

```text
берега западной Африки ниже Северной Африки
Кот-д’Ивуар / Гвинейский залив / красная земля / влажный берег
немного Вьетнама: красная глина, влажная зелень, дороги, грязь
```

Но это не Земля. Это Phoenix7:

```text
океанский порт
красная глина
медное небо
колониальные/имперские блоки
военные реликты
космоопера
Мёртвая Саванна за дорогой
```

## Мир

Первый акт должен начинаться на берегу океана:

```text
Океан
→ Порт Рейчел
→ Таможня
→ рынок и склады
→ дорога вдоль красной глины
→ Форт Заря
→ дом Герды
→ Красный Узел
→ лагерь проводников
→ край Мёртвой Саванны
```

Игрок должен видеть, что мир продолжается, а не стоит в закрытой арене.

## Gameplay identity

Геймплейная смесь:

### Morrowind-like

- медленный темп;
- исследование;
- путешествие;
- разговоры с NPC;
- открытый мир;
- ощущение места;
- странные фракции;
- журнал и задачи;
- не быстрый коридорный action.

### Medal of Honor-like

- старые военные реликты;
- винтовки, пулемёты, револьверы как игровые предметы;
- тяжёлый звук/ритм боя;
- фронтирная военная атмосфера;
- огнестрел как часть мира, не арена-шутер.

### Phoenix7-specific

- фазовый удар рукой;
- элементалы;
- Деймурийцы;
- ордена Зелёных и Синих рыцарей;
- Герда как актовый центр;
- космоопера;
- Мёртвая Саванна;
- странные имперские документы и пустое дело.

## Первый акт — обязательный маршрут

2.5 должна реализовать первый акт:

```text
1. Игрок появляется у океанского Порта Рейчел.
2. Идёт в таможню.
3. Говорит с Риной.
4. Получает направление к Дорожному навесу.
5. Видит дорогу к Форту Заря.
6. Доходит до Форта Заря.
7. Говорит с Ораном Тивом.
8. Получает регистрацию.
9. Находит Герду Гайгерманику.
10. Герда открывает 4 подготовки:
    - дорога;
    - рынок;
    - фракции;
    - контрабанда.
11. Игрок может сдать 4/4.
12. Сава готовит дорогу к Мёртвой Саванне.
```

## Обязательные персонажи

- Рина — таможня Порта Рейчел.
- Оран Тив — канцелярия Форта Заря.
- Герда Гайгерманика — центр первого акта.
- Бродячий рыцарь — дорожный персонаж.
- Торговец красной глиной — рынок.
- Контрабандист — Красный Узел.
- Сава — проводник к Мёртвой Саванне.

## Обязательные боевые элементы

Руки игрока должны быть видны.

Оружие:

```text
1 — кулаки
2 — фазовый удар рукой
3 — бастард
4 — шпага
5 — Кольт 1917
6 — M1 Гаранд
7 — Брен
```

Фазовый удар — короткий удар рукой на ближней дистанции.

Огнестрельное — игровые реликты, не инструкции по реальному оружию.

## Обязательный visual target для 2.5

Когда игрок делает скрин, должно быть понятно:

```text
это атмосферная 3D RPG на странной планете
это берег океана с красной глиной
там есть порт и дорога в саванну
это не пустой тестовый canvas
это можно показать другим людям
```

## Что считать провалом

2.5 не считается успехом, если:

- игрок видит пустой горизонт;
- нет читаемого Порта Рейчел;
- NPC недоступны;
- здания снова цельные коробки;
- нет океана;
- нет дороги в Мёртвую Саванну;
- нет ощущения путешествия;
- игра выглядит как debug map.

## Следующий правильный технический шаг

Не чинить старый raw WebGL кусками.

Нужно собрать новый `game.html` вокруг стабильной 3D-основы:

```text
1. Clean 3D boot.
2. Ocean coast blockout.
3. Port Rachel blockout.
4. Road to Fort Zarya.
5. Fort Zarya + Gerda route.
6. Dead Savannah edge.
7. NPC placement.
8. First Act quests.
9. Hands and arsenal.
10. Visual polish toward Claude build.
```
