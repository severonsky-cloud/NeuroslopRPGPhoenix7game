# Phoenix7 2.0M — Act 1 Living Loop Build

Файл: `act1_loop.html`

## Цель

Собрать не просто красивую сцену, а первый рабочий игровой луп:

```text
создание персонажа
→ выход в мир
→ получение задачи
→ путешествие
→ событие / бой / разговор / улика
→ возвращение к Герде
→ награда
→ открытие следующей подготовки
→ сбор экспедиции
```

## Главный маршрут

1. Порт Рейчел.
2. Рина в таможне.
3. Дорожный навес.
4. Старый маяк.
5. Бродячий рыцарь.
6. Ворота Форта Заря.
7. Оран Тив.
8. Герда.
9. Доска задач Герды.
10. Красный Узел.
11. Лагерь проводников.

## Задачи первого акта

Герда просит собрать 4 подготовки:

- Дорога и навес.
- Рынок и слухи.
- Две фракции: зелёный знак и синий камертон.
- Контрабанда / Красный Узел.

После 4/4 открывается подготовка экспедиции:

```text
Герда → Лагерь проводников → будущий выход к Мёртвой Саванне
```

## Живой мир

В 2.0M добавлено:

- время суток;
- день/ночь через изменение освещения;
- движущийся караван;
- враги, которые реагируют на игрока;
- репутация: Империя, Зелёные, Синие, Караван;
- задачи, которые можно выполнять в разном порядке после открытия Герды;
- rewards loop: сдача задач Герде за XP/кредиты;
- Q — доска задач Герды;
- M — карта акта;
- J — Книга дорог;
- I — инвентарь.

## Запуск

```bash
py -m http.server 8000
```

Открыть:

```text
http://localhost:8000/act1_loop.html
```

## Что проверять

1. Новая игра / быстрый старт.
2. Рина выдаёт задачу на навес.
3. Навес засчитывается.
4. Маяк открывает этап каравана.
5. Рыцарь открывает путь к воротам.
6. Оран выдаёт регистрацию.
7. Герда открывает доску задач.
8. Выполняются рынок, фракции, контрабанда.
9. После 4/4 Герда открывает экспедицию.
10. Проводник завершает акт 1.
