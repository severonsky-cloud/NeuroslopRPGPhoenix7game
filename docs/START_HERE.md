# Phoenix7 — START HERE

## Главный запуск игры

Всегда запускать игру отсюда:

```text
http://localhost:8000/game.html
```

## Локальный запуск

В папке репозитория:

```bash
py -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000/game.html
```

## Что такое остальные HTML

Остальные HTML — лаборатории, прототипы и архивные кандидаты:

- `play.html` — меню всех билдов.
- `act1_arsenal_world.html` — предыдущий арсенальный кандидат.
- `act1_hands_world.html` — руки и фазовый бой.
- `act1_asset_director.html` — staging для ассетов.
- `model_lab.html` — проверка GLB/fallback.
- `audio_lab.html` — проверка звуков.
- `worldmap.html` — карта мира.
- `index.html` — fallback.

## Важное правило с 2.0U

`game.html` — канонический файл. Новые игровые изменения сначала должны попадать туда.

## Исправление зданий

В `game.html` здания больше не являются цельными непроходимыми кубами:

- Таможня имеет дверной проём.
- Канцелярия Орана имеет дверной проём.
- Дом Герды имеет дверной проём.
- NPC стоят внутри доступных помещений.
- Коллизия работает по стенам, а не по всему зданию сразу.

## Текущий игровой кандидат

`game.html` включает:

- первый акт;
- доступные здания с дверями;
- Рину, Орана, Герду и других NPC;
- руки игрока;
- фазовый удар рукой;
- бастард;
- шпагу;
- Кольт 1917;
- M1 Гаранд;
- Брен;
- боезапас и лут у врагов.
