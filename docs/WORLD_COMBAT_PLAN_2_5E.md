# Phoenix7 2.5E — World + Combat Plan

## Главный запуск

```text
START_GAME_WINDOWS.bat
```

или, если сервер уже открыт:

```text
http://localhost:8000/game.html
```

## Цель 2.5E

Сделать билд ближе к настоящей игре, которую можно показать другим:

```text
больше открытого мира
+ понятная география
+ разные биомы
+ NPC по районам
+ улучшенные руки и оружие
+ больше импакта в бою
+ первый акт с маршрутом исследования
```

## Карта мира — новая схема

Мир должен быть не маленькой ареной, а длинным маршрутом-путешествием:

```text
[ОКЕАН / ДОКИ]
      |
      v
[ПОРТ РЕЙЧЕЛ]
  - Таможня / Рина
  - Склад соли
  - Грязный рынок / торговец
  - Старый маяк
      |
      v
[КРАСНАЯ ДОРОГА]
  - Дорожный навес
  - Бродячий рыцарь
  - первые мутанты
      |
      +----> [КРАСНЫЙ УЗЕЛ]
      |       - контрабандист
      |       - красная падаль
      |
      +----> [СТАРАЯ БАТАРЕЯ]
      |       - военная реликтовая зона
      |       - фракционный след
      |
      v
[ФОРТ ЗАРЯ]
  - Канцелярия Орана
  - Дом Герды
  - военные силуэты / башни
      |
      v
[ЛАГЕРЬ ПРОВОДНИКОВ]
  - Сава
      |
      v
[КРАЙ МЁРТВОЙ САВАННЫ]
  - сухие деревья
  - пыльная стена
  - синий след
  - обещание следующего акта
```

## Биомы

### 1. Океанский берег

- влажная красная глина;
- океан слева от стартовой зоны;
- мачты;
- портовые силуэты;
- ящики, соль, доски, склады.

### 2. Порт Рейчел

- рынок;
- таможня;
- старый маяк;
- плотная зона NPC;
- начало первого акта.

### 3. Красная дорога

- открытый путь;
- камни;
- пальмы реже;
- больше расстояния;
- первые враги.

### 4. Форт Заря

- башни;
- тёмные военные блоки;
- канцелярия;
- Герда;
- ощущение военного фронтира.

### 5. Красный Узел

- боковая зона;
- контрабанда;
- красная земля;
- более опасная фауна.

### 6. Старая батарея

- реликт военной эпохи;
- обломки;
- башенки;
- фракционный след.

### 7. Край Мёртвой Саванны

- сухие деревья;
- пустой горизонт;
- пыль;
- переход в следующий акт.

## NPC placement

```text
Рина — внутри таможни
Торговец — грязный рынок
Бродячий рыцарь — красная дорога / старый маяк
Оран Тив — канцелярия Форта Заря
Герда — дом Герды
Контрабандист — Красный Узел
Сава — лагерь проводников
```

## Combat direction

Бой должен быть медленнее и тяжелее, чем арена-шутер:

```text
Morrowind pacing
+ Medal of Honor weapon feel
+ Phoenix7 phase hand strike
```

### Улучшения боя

- руки меньше и ниже;
- оружие меньше закрывает экран;
- recoil после выстрела;
- weapon sway при движении;
- muzzle flash для огнестрела;
- hit marker при попадании;
- небольшой визуальный отклик врага при уроне;
- расход патронов;
- loot после смерти врага.

## Оружие

```text
1 — кулаки
```

## 2.5E Acceptance Test

Билд считается шагом вперёд, если:

```text
1. START_GAME_WINDOWS.bat запускает сервер.
2. game.html открывает игру.
3. Игрок видит более широкий мир, а не маленькую арену.
4. Есть океан, порт, дорога, форт, Красный Узел, батарея, край саванны.
5. M открывает карту с локациями.
6. 5/6/7 показывают более удобное оружие в руках.
7. Выстрел даёт recoil/muzzle flash/hit marker.
8. Врага можно убить и обыскать.
9. Первый акт остаётся читаемым: Рина → навес → Оран → Герда → Сава.
```
