# Phoenix7 v3.0K — Feel Rebuild Plan

## Почему текущий бой ощущается плохо

Системы уже есть, но ощущение слабое, потому что отсутствует главный боевой контур:

```text
нажал кнопку
увидел сильную реакцию оружия
услышал понятный звук
понял состояние оружия
увидел попадание
увидел реакцию цели
получил ясную UI-обратную связь
```

Сейчас многое технически работает, но игрок не чувствует вес.

Главные проблемы:

```text
перезарядка почти невидимая
выстрелы плохо читаются
оружие не имеет достаточно recoil/kick
попадания не имеют hit-pause / hit-flash / stagger
HUD сообщает текстом, но не телом экрана
огнестрел слишком похож на абстрактную функцию
ближний бой тоже нуждается в телесном импакте
NPC стало больше и модели лучше, но сцены ещё пустые
```

## Новый курс

Следующие проходы должны быть не про количество систем, а про качество ощущений.

Правило:

```text
меньше нового контента
больше polish на 3-4 базовых действиях
```

Базовые действия для доведения:

```text
1. выстрел
2. перезарядка
3. попадание
4. ближний удар
5. реакция врага
6. читаемый HUD
```

# v3.0K1 — Gun Feel Pass

## Цель

Сделать огнестрел тяжёлым, старым, грязным и понятным.

## Что сделать

### 1. Recoil Model

Для каждого огнестрела:

```text
camera kick
weapon kick
weapon return curve
horizontal drift
old weapon shake
post-shot recovery
```

Пример направления:

```text
Кольт — короткий резкий подброс
M1 — сухой сильный удар в плечо
Брен — тяжёлая тряска очередью
Дробовик — большой удар и медленное восстановление
Карабин — средний controllable kick
```

### 2. Muzzle Flash

Нужно добавить:

```text
короткая вспышка у ствола
дым/пыль после выстрела
яркий tracer первые метры
маленькое освещение на 0.04 секунды
```

### 3. Tracer Visibility

Текущие трассеры слишком слабые.

Нужно:

```text
толще линия
короче время жизни
ярче середина
слегка тёплый цвет для обычного огнестрела
синий/фиолетовый для фазового
искры на земле/камне/броне
```

### 4. Shot Feedback

При выстреле:

```text
экран микродёргается
weapon model резко уходит назад
прицел расширяется на момент
HUD ammo briefly flashes
звук выстрела имеет низкий удар + короткий щелчок механики
```

# v3.0K2 — Reload Feel Pass

## Проблема

Сейчас перезарядка есть как состояние, но её почти не видно.

## Что сделать

### 1. Reload Stages

Перезарядка должна иметь стадии:

```text
start
open / pull
insert ammo
close / chamber
ready
```

Даже если модели простые, игрок должен видеть движение рук/оружия.

### 2. Weapon-Specific Reload

```text
Кольт — барабан / патроны / закрыть
M1 — пачка / металлический щелчок
Брен — магазин сверху
Дробовик — по одному патрону
Карабин — обойма / затвор
```

### 3. Reload UI

На экране:

```text
крупный статус: RELOADING
полоса прогресса
current loaded / reserve
если осечка: JAMMED — PRESS R
```

### 4. Cancel/Lock

Во время reload:

```text
нельзя стрелять
можно двигаться медленнее
можно отменить переключением оружия позже
```

# v3.0K3 — Hit Feel / Enemy Reaction Pass

## Цель

Каждое попадание должно быть видно без чтения текста.

## Что сделать

### 1. Hit Flash

Враг на 0.06-0.12 сек получает:

```text
короткий светлый flash материала
маленький jerk назад
поворот корпуса
```

### 2. Stagger Classes

```text
small stagger — пистолет / кинжал
medium stagger — винтовка / меч
heavy stagger — дробовик / топор / танковая близкая вспышка
phase stagger — фиолетовый разрыв и задержка
```

### 3. Hit Marker

HUD должен показывать:

```text
маленький белый hit marker
красный/оранжевый критический marker
звук попадания отдельно от выстрела
```

### 4. Body Feedback

Не нужно gore. Нужно читаемо:

```text
отшагнул назад
закрылся
присел
потерял aim
отступает при низком HP
```

# v3.0K4 — Melee Feel Pass

## Проблема

Ближний бой пока механический.

## Что сделать

### 1. Three Attack Types

```text
slash — широкий удар
thrust — укол
heavy — медленный сильный удар
```

### 2. Wind-up and Release

У игрока тоже должен быть:

```text
короткий wind-up
release window
recovery
```

### 3. Trail Improvements

```text
меч — светлая дуга
топор — короткая тяжёлая дуга
дубина — пыльный рывок
штык — узкий прямой след
фаза — фиолетовый разрыв
```

### 4. Impact Lock

При попадании:

```text
0.04 сек микропаузa
камера слегка останавливается
враг получает jerk
звук удара отдельный
```

## Проблема

Интерфейс сейчас информирует, но не помогает чувствовать бой.

## Что сделать

### Combat HUD

```text
ammo widget крупнее
loaded / reserve / condition
reload progress bar
jam warning
weapon name + enchantment
small combat log fading, не огромный текст
```

### Damage Feedback

```text
screen edge flash при попадании по игроку
stamina flash если не хватает выносливости
phase flash если не хватает фазы
hit marker в центре
```

### Inventory UI Later

Инвентарь пока оставить системным.
Сначала нужно довести бой.

# v3.0L — NPC Density / Scene Composition

После feel-pass можно расширять мир.

## Цель

Побольше NPC, но не просто толпа.

## Что добавить

```text
малые группы у дорог
посты у перекрёстков
крестьяне у деревень
ремонтники и торговцы у форта
караванщики с охраной
патрули на петлях
фазовые маги у аномалий
элементали у своих караванов
```

```text
порт/форт: плотнее
дороги: редкие, но запоминающиеся группы
пустошь: мало, но опаснее
лес: силуэты и скрытые NPC
```

# Критерий готовности v3K

Сегмент считается успешным, если игрок без чтения текста понимает:

```text
оружие выстрелило
оружие перезаряжается
оружие дало осечку
попал или промахнулся
враг получил удар
враг отступил/ошеломлён
патроны заканчиваются
состояние оружия плохое
```

Это следующий главный рубеж проекта.
