# Phoenix7 2.0G — новая карта открытого мира

Файл карты: `worldmap.html`.

Это не старая карта, а новая производственная карта, использующая старые наброски только как референс.

## Главная структура

Запад → Восток:

1. **Синее море / Порт Рейчел** — стартовая область.
2. **Синий Шельф** — обучение, первые NPC, дорога.
3. **Форт Заря** — центральный хаб первого акта.
4. **Красный Узел / Тёплые каналы** — контрабанда, красная глина, первые серьёзные опасности.
5. **Мёртвая Саванна** — второй акт, Гравийная чума, чёрные элементали, Жужжары.

Север / Юг:

- Север: монастырские дороги, проводники, фазовые зоны, Пепельные ворота Цикла.
- Юг: Зелёный разлом Системных Ошибок, южные ритуальные зоны.
- Северо-восток/юго-восток: Синий Орден и дальнейший выход к резонансным областям.

## Стартовая цепочка

```text
Порт Рейчел → Таможня → Дорожный навес → Старый маяк → Ворота Форта Заря → Оран Тив → Герда
```

## Первые игровые узлы

- Порт Рейчел — старт.
- Таможня — объяснение пустого дела.
- Дорожный навес — первая задача Герды.
- Старый маяк — ориентир и фракционный намёк.
- Форт Заря — hub.
- Канцелярия Орана — регистрация.
- Дом Герды — первый актовый quest hub.
- Грязный рынок — торговля, слухи, будущая контрабанда.

## Фракционные ветки

- Придорожный знак.
- Южный зелёный разлом.
- Монастырь Системных Ошибок.
- Темы: сломанные терминалы, богиня системных ошибок, неверные старты, память мира.

- Монастырь Звука.
- Темы: резонанс, музыка, звуковые ритуалы, Great Concert.

### Империя

- Форт Заря.
- Канцелярия.
- Рынок.

### Мёртвая Саванна

- Вход во второй акт.

- Костяной маяк Жужжар.
- Гравийная чума.

## Как использовать карту в производстве

1. `worldmap.html` — визуальный атлас.
2. `real3d.html` — текущий 3D срез.
3. Следующая задача: перенести узлы `worldmap.html` в data-driven JSON.
4. Потом делать сцену не всей карты сразу, а чанками:
   - Port Rachel chunk.
   - Blue Shelf Road chunk.
   - Fort Zarya chunk.
   - Green Shrine chunk.
   - Blue Shrine chunk.

## Почему так

Старые карты были перегружены и часто ломали старт. Новая карта строится вокруг понятного маршрута игрока и расширяется слоями, а не хаотично.
