# Phoenix7 2.4 — Ready Build Status

## Главный запуск

Всегда запускать игру отсюда:

```text
http://localhost:8000/game.html
```

Локально:

```bash
py -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000/game.html
```

## Что считается главным билдом

`game.html` — канонический вход в игру.

Старые HTML больше не считаются главным билдом. Они остаются как лаборатории и архивные кандидаты.

## Что сейчас есть в рабочем билде

### Мир

- Порт Рейчел / стартовая зона.
- Дорога к Форту Заря.
- Канцелярия Орана.
- Дом Герды.
- Дорожный навес.
- Старый маяк.
- Грязный рынок.
- Красный Узел.
- Лагерь проводников.

### Здания и двери

Проблема цельных зданий исправляется в каноническом направлении:

- NPC не должны быть замурованы в непроходимых кубах.
- Таможня, канцелярия и дом Герды должны иметь входы.
- Коллизия должна быть по стенам, а не по всему зданию.

### Персонажи

В мире введены:

- Рина — таможенный клерк Порта Рейчел.
- Оран Тив — писарь/регистратор Форта Заря.
- Герда Гайгерманика — главный квестовый центр первого акта.
- Бродячий рыцарь.
- Торговец красной глиной.
- Контрабандист у Красного Узла.
- Проводник Сава.

### Первый акт

Текущая структура первого акта:

```text
Порт Рейчел
→ Рина
→ дорожный навес
→ Форт Заря
→ Оран
→ Герда
→ подготовка Герды:
   - дорога
   - рынок
   - фракции
   - контрабанда
→ проводник / путь к Мёртвой Саванне
```

### Руки и бой

Есть видимые руки игрока и боевые режимы:

```text
1 — кулаки
2 — фазовый удар рукой
3 — бастард
4 — шпага
5 — Кольт 1917
6 — M1 Гаранд
7 — Брен
```

Фазовый удар рукой — короткий ближний удар, а не дальний файрбол.

### Боезапас и лут

Есть отдельный боезапас:

- к Кольту 1917;
- к M1 Гаранд;
- к Брену.

У врагов разные loot tables:

- дорожный мутант;
- песчаная падаль;
- красная падаль;
- фазовое эхо.

Лут может дать:

- боезапас;
- кредиты;
- жетон дороги;
- канальную бирку;
- фазовый осадок.

## Управление

```text
WASD — движение
Shift — бег
мышь — камера / направление
ЛКМ или Space — атака
E — говорить / обыскать
R — блок / фазовый фокус
1 — кулаки
2 — фазовый удар рукой
3 — бастард
4 — шпага
5 — Кольт 1917
6 — M1 Гаранд
7 — Брен
J — журнал
I — инвентарь
Q — задачи Герды
F1 — тестовый телепорт
Esc — закрыть панель / выйти из pointer lock
```

## Что НЕ трогать как главный запуск

Эти файлы не являются главным билдом:

- `act1_loop.html`
- `act1_integrated.html`
- `act1_sound_integrated.html`
- `act1_asset_director.html`
- `act1_hands_world.html`
- `act1_arsenal_world.html`
- `model_lab.html`
- `audio_lab.html`
- `worldmap.html`
- `game_stable_2d.html`

Их можно использовать только для сравнения, референса и восстановления отдельных механик.

## Что тестировать сейчас

Минимальный тест:

1. Открыть `http://localhost:8000/game.html`.
2. Нажать быстрый старт или новую игру.
3. Проверить, что игра запускается без чёрного экрана.
4. Дойти до таможни и поговорить с Риной.
5. Проверить входы в здания.
6. Проверить оружие 1–7.
7. Убить врага и обыскать его через `E`.
8. Проверить журнал `J`, инвентарь `I`, задачи `Q`.

## Ближайшая цель

Версия 2.5 должна стать не набором прототипов, а одним играбельным vertical slice:

```text
единый запуск
+ 3D-мир
+ двери/интерьеры
+ руки
+ арсенал
+ NPC
+ первый акт
+ звук
+ модельный fallback
+ сохранение
```
