# Phoenix7 v3.0D — Action RPG Combat System

## Цель

Phoenix7 должен быть action RPG, а не просто прогулочным прототипом.

Боевая система должна соединять:

```text
Morrowind-style RPG progression
+ first-person action combat
+ фазовая магия
+ инвентарь с куклой персонажа
+ живой мир и фракционные конфликты
```

---

## Основные элементы

### 1. Инвентарь

Инвентарь открывается клавишей:

```text
I
```

Инвентарь ставит игру на паузу.

В инвентаре есть:

```text
кукла персонажа / paper doll
слоты брони
слоты экипировки
левый набор оружия
правый набор оружия
фазовая рука
список предметов
вес
броня
```

### 2. Кукла персонажа

Слоты:

```text
head
chest
hands
legs
boots
amulet
leftHand
rightHand
spellHand
```

### 3. Левая и правая рука

В текущей версии левая/правая рука — это не симуляция двух рук одновременно.

Это два быстрых набора оружия:

```text
leftHand  — основной набор
rightHand — запасной набор
```

Переключение:

```text
Tab
```

Идея: игрок сам выбирает, например:

```text
leftHand: Бастард
rightHand: Кольт 1917
```

или:

```text
leftHand: Шпага
rightHand: M1 Гаранд
```

### 4. Контекстное окно

Правая кнопка мыши:

```text
ПКМ
```

Открывает контекстное меню и ставит игру на паузу.

Контекстное меню даёт доступ к:

```text
инвентарю
персонажу
фазовой магии
карте
журналу
смене набора оружия
прицельному режиму
```

### 5. Прицел / мушка

Клавиша:

```text
V
```

Один раз нажать — включить прицельный режим.
Ещё раз нажать — вернуться к обычному обзору.

Прицел не нужно удерживать.

В прицельном режиме:

```text
камера сужает FOV
оружие ближе к центру
появляется более строгая мушка
свей уменьшается
```

## RPG-система

### Основные параметры

```text
Сила
Ловкость
Выносливость
Интеллект
Воля
Восприятие
```

### Навыки

```text
Клинки
Рукопашный бой
Огнестрел
Броня
Атлетика
Уклонение
Фазовая магия
Выживание
Речь
```

### Принцип прокачки

Навык растёт, когда игрок использует его:

```text
ударяешь мечом     → Клинки
стреляешь          → Огнестрел
бежишь             → Атлетика
используешь фазу   → Фазовая магия
говоришь с NPC     → Речь
получаешь удары    → Броня
```

Также навыки можно повышать:

```text
у учителей
через квестовые награды
через редкие книги/память/фазовые следы
```

### Уровень

На новый уровень игрок получает:

```text
2 очка основных параметров
1 очко фазового перка
```

Дополнительные очки фазовых перков можно получать:

```text
за редкие фазовые аномалии
за обучение у Гильдии Фазовых Магов
за квесты чёрных/синих элементалей
за прохождение опасных зон без смерти
за нахождение фазовых якорей
за события Мёртвой Саванны
```

## Фазовая магия

Фазовая магия выбирается в руку.

Окно:

```text
P
```

Способности на хотбаре:

```text
Z
X
C
F
```

### Ветки фазовой магии

```text
Фазовая атака
Фазовая защита
Фазовое путешествие
```

### Примеры умений

Фазовая атака:

```text
Фазовый удар
Конус сдвига
Разрыв контура
```

Фазовая защита:

```text
Фазовый заслон
Эхо-кожа
Расщеплённая секунда
```

Фазовое путешествие:

```text
Короткий сдвиг
Чувство дороги
Якорь Саванны
```

## Бой

### Типы оружия

```text
кулаки
фазовая рука
мечи
шпаги
огнестрельное оружие
```

### Клавиши оружия

```text
1 — кулаки
2 — фазовая рука
3 — бастард
4 — шпага
5 — Кольт 1917
6 — M1 Гаранд
7 — Брен
```

### Ближний бой

Ближний бой должен использовать сектор удара, а не точечный raycast.

Проверяется:

```text
дистанция до врага
сектор перед игроком
тип оружия
навык
выносливость
фазовые эффекты
```

### Фазовая атака

Фазовая атака тратит фазовый ресурс.

Если фазы мало — атака не сработает.

Фазовые перки меняют:

```text
урон
сектор
стоимость
эффекты
дальность
```

### Огнестрел

Огнестрельное оружие использует:

```text
патроны
навык Огнестрел
прицел V
свей
отдачу
дистанцию
```

Это игровая художественная система, не реалистичный симулятор.
