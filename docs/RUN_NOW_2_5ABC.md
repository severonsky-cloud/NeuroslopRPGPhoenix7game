# Phoenix7 2.5A/B/C — Run Now

## Главный запуск

```text
http://localhost:8000/game.html
```

## Локальный запуск

В папке репозитория:

```bash
py -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000/game.html
```

## Что сейчас должно открыться

Стартовый экран:

```text
Phoenix7 2.5A/B/C — clean 3D boot
```

После `Быстрый старт`:

- 3D WebGL-сцена на Three.js;
- океан слева/сзади от Порта Рейчел;
- красная глина;
- дорога;
- Порт Рейчел;
- таможня с входом;
- Рина внутри;
- рынок;
- дорога к Форту Заря;
- канцелярия Орана;
- дом Герды;
- край Мёртвой Саванны;
- враги;
- руки игрока и оружие.

## Первый тест

1. Открыть `game.html`.
2. Нажать `Быстрый старт`.
3. Кликнуть в экран, чтобы поймать мышь.
4. Идти к Рине в таможню.
5. Подойти к Рине и нажать `E`.
6. Проверить, что диалог открывается.
7. Нажать `1–7`, проверить смену оружия.
8. Найти врага и атаковать.
9. Обыскать врага через `E` после смерти.

## Если не запускается

Если видишь красный JS ERROR:

- пришли скрин;
- особенно строку ошибки;
- не открывай другие HTML как замену.

Чиним только:

```text
game.html
src/game/main.js
```

## Что уже проверено структурно

`game.html` теперь:

- имеет canvas `#game`;
- имеет HUD с нужными ID;
- имеет стартовый экран;
- грузит только `src/game/main.js`;
- больше не содержит старый raw WebGL-код;
- показывает красный overlay, если модуль не загрузился.
