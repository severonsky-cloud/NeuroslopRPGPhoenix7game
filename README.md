# Phoenix7 2.0B — Production Core

Новый чистый билд Phoenix7 на Three.js. Это не продолжение старого монолитного HTML 1.6, а новый фундамент для дальнейшего производства.

## Что уже есть

- Честная 3D-сцена на Three.js/WebGL.
- Старт в **Порту Рейчел**.
- Дорога Синего Шельфа к **Форту Заря**.
- Первый квест: **Порт Рейчел: пустое дело**.
- NPC: портовый рабочий, таможенный клерк, солдат у дороги, бродячий рыцарь, зелёный послушник, синий слушатель, писарь Оран Тив, Герда, торговец.
- Живой дорожный слой: малый караван и патрули двигаются по дорогам.
- Книга дорог по `J`.
- Карта по `M`.
- Pointer Lock / мышь от первого лица.
- Процедурные звуки Web Audio как fallback.
- Production docs: `docs/PRODUCTION_PLAN_2_0.md`, `docs/ASSET_PIPELINE.md`.
- Data outline: `src/data/world-outline.json`.
- Никакого старого старта **Гравийный рубеж** и никакого **Приказ №7**.

## Как запустить

Лучше через GitHub Pages или любой static hosting.

Локально:

```bash
python3 -m http.server 8000
```

Открыть:

```text
http://localhost:8000/
```

Можно открыть и как обычный HTML, но для модульного JS и внешнего Three.js лучше использовать локальный сервер.

## Управление

- `WASD` — ходьба
- `Shift` — быстрее
- мышь после клика по экрану — обзор
- `E` — говорить с ближайшим NPC
- `J` — Книга дорог
- `M` — карта
- `Esc` — снять pointer lock / закрыть окно

## Важное решение

2.0 не переносит старые script-слои 1.6. Старые hotfix-ветки ломали стартовую цепочку и возвращали старый квест. Новый билд начинается с чистого runtime.
