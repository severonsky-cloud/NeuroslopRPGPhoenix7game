# Phoenix7 — Canonical Playable Build

## Главный запуск игры

Всегда запускать игру отсюда:

```text
http://localhost:8000/game.html
```

Локально в папке репозитория:

```bash
py -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000/game.html
```

`play.html` теперь только меню билдов. Старые `act1_*.html`, `model_lab.html`, `audio_lab.html` и прочие файлы — лаборатории/архив, а не главный запуск.

## Управление

```text
WASD — движение
Shift — бег
мышь — камера / направление
ЛКМ или Space — атака
E — говорить / обыскать
R — блок / фазовый фокус
1 — кулаки
2 — фазовый удар рукой
3 — бастард
4 — шпага
5 — Кольт 1917
6 — M1 Гаранд
7 — Брен
J — журнал
I — инвентарь
Q — задачи Герды
F1 — тестовый телепорт
Esc — закрыть панель / выйти из pointer lock
```

## Цель версии 2.5

2.5 должна быть первым цельным playable vertical slice:

```text
один запуск: game.html
+ доступные здания и двери
+ NPC внутри и снаружи
+ первый акт Герды
+ руки игрока
+ арсенал
+ фазовый удар
+ враги и лут
+ звук
+ модельный fallback
+ сохранение
```

## Правило разработки

Новые игровые правки сначала идут в:

```text
game.html
```

Лаборатории можно использовать как доноров, но игрок не должен бегать по разным HTML-файлам.
