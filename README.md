# Phoenix7 2.0A — Real Three.js Rebuild

Это новый чистый билд Phoenix7, созданный как отдельный фундамент вместо продолжения старого монолитного HTML 1.6.

## Что внутри

- Честная 3D-сцена на Three.js.
- Старт в Порту Рейчел.
- Дорога к Форту Заря.
- Первый квест: **Порт Рейчел: пустое дело**.
- NPC: портовый рабочий, таможенный клерк, солдат у дороги, писарь Оран Тив, Герда-заглушка.
- Книга дорог по `J`.
- Карта по `M`.
- Pointer Lock / мышь от первого лица.
- Процедурные звуки Web Audio, чтобы билд работал без внешних WAV.
- Никакого старого старта **Гравийный рубеж** и никакого **Приказ №7**.

## Как запустить

Лучше через GitHub Pages или любой static hosting.

Локально:

```bash
python3 -m http.server 8000
```

Открыть:

```text
http://localhost:8000/
```

Можно открыть и как обычный HTML, но для модульного JS и внешнего Three.js лучше использовать локальный сервер.

## Управление

- `WASD` — ходьба
- `Shift` — быстрее
- мышь после клика по экрану — обзор
- `E` — говорить с ближайшим NPC
- `J` — Книга дорог
- `M` — карта
- `Esc` — снять pointer lock / закрыть окно

## Важное решение

Этот билд специально не переносит старые script-слои 1.6, потому что они ломали стартовую ветку и постоянно возвращали старый квест. 2.0 начинается с чистого runtime.
